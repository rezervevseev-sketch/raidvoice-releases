# Changelog

## v0.3.4 (2026-01-30)

### Исправления
- **Счётчик онлайн** — теперь корректно обновляется при автоматическом выходе неактивных пользователей
- **Отображение участников** — исправлено "1 участник" в пустой комнате
- **Выход из клуба** — гости клуба теперь возвращаются на экран входа в клуб, а не на обычный логин
- **Стабильность** — улучшена надёжность процесса выхода из аккаунта

---

## v0.3.3 (2026-01-30)

### Что нового
- **Hardware ID авторизация** — автоматический вход на бездисковых ПК
  - При перезагрузке ПК сессия восстанавливается автоматически
  - Работает для клубов с PXE/iSCSI загрузкой
  - Привязка к уникальному идентификатору железа
- **Улучшенная инициализация** — экран загрузки во время проверки авторизации

---

## v0.3.2 (2026-01-29)

### Исправления
- **Socket интеграция** — счётчик участников теперь обновляется в реальном времени при входе/выходе
- Исправлено отображение "1 участник" когда комната пустая
- Исправлена возможность создания LFG для гостей клуба

---

## v0.3.1 (2026-01-29)

### Что нового
- **Система клубов** — MVP для компьютерных клубов
  - Авторизация через номер ПК
  - Отдельный сайдбар с участниками клуба
  - Приглашения в комнату одним кликом
  - До 10 участников в комнате (для обычных пользователей — до 6)
- **Улучшения интерфейса**
  - Зелёная подсветка комнат с участниками
  - Минималистичный дизайн подсказок
- **Улучшения в реальном времени**
  - Мгновенное обновление статуса друзей (в сети/отошёл)
  - Счётчик участников обновляется даже когда вы не в комнате

### Исправления
- Исправлен критический баг с пустым экраном после входа
- Исправлена проблема с длинными именами ПК в клубах
