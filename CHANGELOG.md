# Changelog

## v0.3.1 (2026-01-29)

### Что нового
- **Система клубов** — MVP для компьютерных клубов
  - Авторизация через номер ПК
  - Отдельный сайдбар с участниками клуба
  - Приглашения в комнату одним кликом
  - Возможность приостановить доступ к клубу
- **Увеличен лимит комнат** — теперь можно быть в 6 комнатах одновременно (было 5)
- **Улучшения интерфейса**
  - Зелёная подсветка комнат с участниками
  - Минималистичный дизайн подсказок
- **Улучшения в реальном времени**
  - Мгновенное обновление статуса друзей (в сети/отошёл)
  - Счётчик участников обновляется даже когда вы не в комнате

### Исправления
- Исправлен критический баг с пустым экраном после входа
- Исправлена проблема с длинными именами ПК в клубах

## v0.2.15 (2026-01-21)

### Что нового
- **Улучшена регистрация** — теперь показывается уведомление "Аккаунт создан! Проверьте почту для подтверждения (письмо может быть в спаме)"
- **Система обновлений** — проверка версии только при запуске, не прерывает игру

### Исправления
- Исправлен критический баг при регистрации (не показывалось уведомление)
- Увеличено время показа уведомления о регистрации (6 секунд)

---

## v0.2.14 (2026-01-20)

### Что нового
- **Закрытие приватных комнат** — владелец теперь может закрыть комнату через контекстное меню
  - ПКМ по приватной комнате → "Закрыть комнату"
  - Все участники отключаются
  - Комната удаляется у всех

---

## v0.2.13 (2026-01-20)

### Что нового
- **Заглушение участников** — владелец комнаты может заглушить микрофон участника
  - Оранжевая кнопка рядом с кнопкой "выгнать"
  - Участник получает уведомление "X заглушил ваш микрофон"
  - Мягкий мут — участник может включить микрофон обратно
  - Полезно когда кто-то забыл выключить микрофон
- Улучшена подсказка для кнопки "выгнать" — теперь объясняет что выгнанный не сможет вернуться

### Исправления
- Исправлен баг LFG "+1 игрока" — пост больше не закрывается сразу после создания
- Фильтр игр теперь с поиском и автодополнением (вместо выпадающего списка)
- Исправлено удаление приватной комнаты при подключении
