# RaidVoice

**Голосовой чат для игр. Серверы в России.**

## Скачать

[**Скачать последнюю версию**](https://github.com/rezervevseev-sketch/raidvoice-releases/releases/latest)

## Возможности

- Голосовая связь с низким пингом
- LFG — поиск тиммейтов за минуту
- Личные и приватные комнаты
- Push-to-Talk и горячие клавиши
- Шумоподавление

## Системные требования

- Windows 10/11 (x64)

## Установка

1. Скачайте установщик
2. Запустите файл
3. При предупреждении Windows SmartScreen: нажмите «Подробнее» → «Выполнить в любом случае»
